Programa para controle de carros e tarefas atreladas a eles. Idealizado para o controle de uma funilaria.

Feito em java com frontend Swing e banco de dados MySQL.
Atividade realizada a partir do projeto do curso Lógica de programação e algoritmos III da ProWay EAD.

Padrão MVC, orientado a objetos e orientado a eventos, IDE NetBeans.
